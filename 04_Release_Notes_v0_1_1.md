# Release Notes - v0.1.1

## Purpose of this release

GC Open v0.1.1 is a corrected publishable working release of the Genesis Coherence method pack.

The aim of this release is to publish a usable minimum method with one real example pack and a small set of reusable release documents.

## What this release contains

- generic core release documents
- one internal documentation pilot pack
- one short license and usage note
- one pack overview
- one top-level README

## What is stable enough to use now

The following parts are stable enough for limited practical use:
- generic role map
- generic RACI template
- internal documentation pilot pack
- AI use logging
- review and evidence tracking
- basic release documentation for publication

## What is still draft

The following areas should still be treated as draft or subject to adjustment:
- broader public website framing beyond this pack
- sector-specific adoption guidance
- additional example packs outside internal documentation
- any future control-backbone standards beyond this release layer

## Expected changes in a later version

Likely areas for a later version:
- one or more additional example packs
- tighter naming and versioning conventions
- clearer role guidance across examples
- additional adoption guidance
- refinement based on practical use feedback

## Release position

This release is intentionally modest. It should not be presented as a finished platform or universal governance framework.
It is a bounded, publishable method pack.
