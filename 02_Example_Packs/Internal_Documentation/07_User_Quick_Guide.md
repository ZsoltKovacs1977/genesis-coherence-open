# User Quick Guide

## Use AI only for approved support work
Allowed examples:
- draft internal notes
- summarise approved material
- reformat existing text
- prepare first-pass process wording
- create handoff or meeting summary drafts

Not allowed without separate approval:
- final decisions
- use of restricted or confidential material in unapproved tools
- release of unchecked AI output
- use of AI in a task outside the approved pilot boundary

## User responsibilities
- define the task clearly
- use only approved source material
- review or submit for review before use
- log the work
- escalate when unsure
