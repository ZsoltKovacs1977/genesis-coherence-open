# Internal Documentation Pilot RACI

This RACI assigns minimum ownership for the pilot. One task should have one clear approver.

| Task | A | R | C | I |
|---|---|---|---|---|
| Approve pilot boundary | Sponsor / Team Lead | Admin / Coordinator | Legal / Data Reviewer if needed | Team |
| Nominate pilot roles | Sponsor / Team Lead | Admin / Coordinator | Legal / Data Reviewer if needed | Team |
| Define permitted use cases | Sponsor / Team Lead | Admin / Coordinator | Reviewer; Legal / Data Reviewer if needed | Team |
| Run AI-assisted task | Task Owner | Task Owner | Reviewer | Admin / Coordinator |
| Review output | Reviewer | Reviewer | Task Owner | Admin / Coordinator |
| Approve release or operational use | Approver | Task Owner | Reviewer | Admin / Coordinator |
| Maintain AI use log | Sponsor / Team Lead | Admin / Coordinator | Users; Reviewers | Team |
| Produce weekly scorecard | Sponsor / Team Lead | Admin / Coordinator | Reviewer | Team |
| Final pilot review | Sponsor / Team Lead | Admin / Coordinator | Reviewer; Legal / Data Reviewer if needed | Team |
