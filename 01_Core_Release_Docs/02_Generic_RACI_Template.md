# Generic RACI Template

Use this template to assign minimum ownership for a bounded AI-assisted workflow.

## Role columns
- **Sponsor** - authorizes scope, boundaries, and oversight expectations.
- **Task Owner** - performs the task or ensures the task is performed correctly.
- **Reviewer** - checks quality, completeness, and fitness for purpose before approval.
- **Approver** - accepts or rejects release or operational use.
- **Admin / Coordinator** - maintains records, tracks status, and supports workflow administration.
- **Legal / Data Reviewer** - provides specialist review where data, privacy, or legal concerns require it.

## Notes

- Keep the template lean. Add rows only when they improve clarity.
