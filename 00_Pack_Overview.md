# GC Open v0.1.1 Publishable Pack

This pack combines the release-layer documents for Genesis Coherence Open v0.1.1 with one non-university example pack for internal documentation work.

## Included

1. Core release documents
2. Internal Documentation Pilot Pack
3. License and usage note
4. Release notes for v0.1.1

## Intended use

Use this pack to:
- explain the method in a neutral way
- assign roles consistently across bounded use cases
- run a small internal documentation pilot
- publish a minimal, reusable working release

## Scope note

This pack is a practical release bundle. It does not claim to be a full enterprise AI governance system.
